# @erye/create-vaf

[![npm](https://img.shields.io/npm/v/@erye/create-vaf)](https://www.npmjs.com/package/@erye/create-vaf)

生成 [`vaf` 管理平台应用](https://www.npmjs.com/package/@erye/vaf) 的模板源码。

## 下载模板

```bash
# 在当前工作空间下生成
$ npm create @erye/vaf

# 在当前工作空间的helloworld目录下生成
$ npm create @erye/vaf helloworld
```
